How to implement a new directory type

 

1. Create a subclass of DirectoryConfigurator, ensuring that after configuring the DirectoryConfiguration subclass has been passed to Domain

2. Create a subclass of DirectoryConfiguration to store the configuration information

3. Create a subclass of DirectorySPI to provide the directory services

4. Add an entry to PN_DIRECTORY_PROVIDER_TYPE to describe the directory service and name the classes created above
