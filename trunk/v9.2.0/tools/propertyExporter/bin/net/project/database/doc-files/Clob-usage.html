How to use the Clob object to manipulate Clob data

 

02/25/2003 - New Mechanism

 

See net.project.ClobHelper Usage Documentation for details on the new mechanism.

 

Old Mechanism

 

This mechanism was originally conceived when CLOB columns were maintained in entirely separate tables and avoided the need to explicitly code two separate insert or update statements.

 

Creating a new Clob row

 

Either:

Clob clob = new Clob();    // Own transaction

or

Clob clob = new Clob(db);  // Use specified transaction

 

clob.setTableName(“pn_some_clob_table”);

clob.setIDColumnName(“object_id”);

clob.setDataColumnName(“clob_field”);

clob.setData(“The data to store in the CLOB field”);

clob.generateID();   // Optionally manually generate the clob id

clob.store();        // Store clob, generating id if necessary

newID = clob.getID();

clob.commit();

 

 

Reading an existing Clob row (in read-only mode)

 

Either:

Clob clob = new Clob();    // Own transaction

clob.setTableName(“pn_some_clob_table”);

clob.setIDColumnName(“object_id”);

clob.setDataColumnName(“clob_field”);

clob.setID(“some id value”);

clob.loadReadOnly();       // Performs a rollback!

String data = clob.getData();

 

 

Reading an existing Clob row (for update)

 

Either:

Clob clob = new Clob();    // Own transaction

or

Clob clob = new Clob(db);  // Use specified transaction

 

clob.setTableName(“pn_some_clob_table”);

clob.setIDColumnName(“object_id”);

clob.setDataColumnName(“clob_field”);

clob.setID(“some id value”);

clob.loadReadWrite();      // Update in own transaction

clob.clearData();

clob.setData(“Some data”);

clob.appendData(“Some more data”);

clob.store();    // Store in current transaction

clob.commit();

 

Deleting a Clob row

 

Either:

Clob clob = new Clob();       // Own transaction

or

Clob clob = new Clob(db);     // Use specified transaction

 

clob.setTableName(“pn_some_clob_table”);

clob.setIDColumnName(“object_id”);

clob.setDataColumnName(“clob_field”);

clob.setID(“some id value”);

clob.remove();

clob.commit();
