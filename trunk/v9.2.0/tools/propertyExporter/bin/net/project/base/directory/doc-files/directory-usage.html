How to use the Directory classes

 

Authentication

 

Assume that directoryProviderType is an instantiated DirectoryProviderType object and directoryConfiguration is an instantiated DirectoryConfiguration object.

 

Directory directory = Directory.getInstance(directoryProviderType, directoryConfiguration);

directory.setAuthenticationContext(new AuthenticationContext(“scott”, “tiger”));

directory.authenticate();

 

if (directory.isAuthenticated()) {

    // proceed

} else {

    // Invalid username or password

}
