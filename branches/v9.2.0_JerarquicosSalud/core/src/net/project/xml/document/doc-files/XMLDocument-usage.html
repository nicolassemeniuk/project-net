How to use XMLDocument

 

Important methods

 

*         startElement(String name) – Starts an element

*         endElement() – Ends the current element

*         addElement(String name, Object value) – Adds an element (Starts and ends)

*         addAttribute(String name, Object value) – Adds an attribute to the current element

*         addValue(Object value) – Adds a value to the current element

*         addComment(String text) – Adds a comment

 

In all cases, any special characters are escaped (for example, ‘, <, >, & etc.).

 

 

Example

 

Code:

 

XMLDocument doc = new XMLDocument();

 

doc.startElement("Person");

 

doc.addElement("FirstName", "John");

doc.addElement("LastName", "Doe");

doc.addComment("The DisplayName may differ from FirstName and LastName");

doc.addElement("DisplayName", "John Doe");

doc.addElement("Biography", "This may contain 'special' characters like <, >, &");

 

doc.startElement("Birthday");

doc.addAttribute("year", "1975");

doc.addAttribute("month", "07");

doc.addAttribute("day", "09");

doc.addValue("July 9th, 1975");

doc.endElement();

 

doc.endElement();

 

Result:

 

<?xml version="1.0" ?>

<Person>

    <FirstName>John</FirstName>

    <LastName>Doe</LastName>

    <!-- The DisplayName may differ from FirstName and LastName -->

    <DisplayName>John Doe</DisplayName>

    <Biography>This may contain &apos;special&apos; characters like &lt;, &gt;, &amp;</Biography>

    <Birthday year="1975" month="07" day="09">July 9th, 1975</Birthday>

</Person>
