How to use the Money class

 

Displaying

 

To display a money value read-only, use the taglib:

 

<output:money money=”<%=project.getCost()%>” currency=”<%=project.getDefaultCurrency()%>” />

 

For the money value “9.99 USD” it would be displayed as:

 

9.99 USD

 

Inputting

 

To provide an input field for entering a money value, use the taglib:

 

<input:money name=”cost” money=”<%=project.getCost()%>” currency=”<%=project.getDefaultCurrency()%>” />

 

If the money’s currency is the same as the specified currency and the money is “9.99 USD” the result is:

<input type=”text” name=”cost_value” value=”9.99”><input type=”hidden” name=”cost_currencyCode”>&nbsp;USD

 

If the money’s currency differs from the specified currency (say “GBP”) the result is a selection list that allows the user to choose between the current currency and the specified currency.  For example, for “9.99 USD”:

 

<input type=”text” name=”cost_value” value=”9.99”>&nbsp;

<select name=”cost_currencyCode”>

<option value=”USD”>USD</option>

<option value=”GBP”>GBP</option>

</select>

 

These fields can be used for parsing a new Money value.
