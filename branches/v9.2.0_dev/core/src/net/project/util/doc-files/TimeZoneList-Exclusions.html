TimeZone List Exclusions

 

Removed Items

 

This document describes the method by which we exclude certain TimeZone selections from the list of available timezones.

 

Any time zones with a specified “Old Name” are eliminated.  We are assuming the Java provides the new name.  Any time zones listed as “Obsolete” are eliminated.
